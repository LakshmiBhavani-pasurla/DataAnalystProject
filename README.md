# Customer Insights & Sales Analytics Dashboard

## End-to-End Data Analytics & Business Intelligence Project

An end-to-end Data Analytics project developed using **Python, PostgreSQL, SQL, Jupyter Notebook, and Power BI** to analyze customer shopping behavior and generate actionable business insights through interactive dashboard reporting.

#  Project Overview

The Customer Insights & Sales Analytics project focuses on analyzing customer purchasing behavior, sales performance, revenue contribution, subscription trends, and demographic-based shopping patterns.

The project demonstrates a complete Business Intelligence workflow including:
- data preprocessing,
- exploratory data analysis,
- SQL business analytics,
- KPI generation,
- and interactive dashboard visualization.

The final Power BI dashboard helps businesses understand:
- customer demographics,
- purchasing trends,
- product category performance,
- subscription behavior,
- and sales analytics.

#  Project Objectives

The primary objectives of this project are:

- Analyze customer shopping behavior
- Identify revenue-generating product categories
- Segment customers based on purchase history
- Compare subscriber vs non-subscriber behavior
- Analyze revenue contribution across age groups
- Develop interactive KPI dashboards
- Support data-driven business decision-making

# 🧹 Data Preprocessing & Cleaning

Data preprocessing and cleaning were performed using Python and Pandas in Jupyter Notebook before loading the dataset into PostgreSQL.

## Data Cleaning Activities

- Loaded dataset using pandas
- Checked dataset structure and data types
- Identified missing values and inconsistent records
- Handled missing review ratings using median value imputation
- Standardized column names into snake_case format
- Created age_group segmentation for demographic analysis
- Removed redundant columns
- Validated cleaned records before database loading


# PostgreSQL Database Implementation

The cleaned dataset was imported into PostgreSQL for SQL-based business analysis.

## Database Name
